# Сервіс замовлення сайтів та конспектів

Сучасний веб-сайт для замовлення різних послуг: розробка сайтів, написання конспектів, програмування, курсові роботи.

## Технології

- **HTML** - структура сторінок
- **CSS** - стилізація та дизайн
- **JavaScript** - інтерактивність та валідація форм
- **Python (Flask)** - backend для обробки замовлень
- **C++** - модуль для розрахунку цін та обробки даних

## Встановлення

### 1. Встановлення Python залежностей

```bash
pip install -r requirements.txt
```

### 2. Компіляція C++ модуля (опціонально)

Для Windows:
```bash
build.bat
```

Для Linux/Mac:
```bash
g++ -shared -fPIC order_processor.cpp -o order_processor.so
```

Якщо C++ модуль не скомпільовано, система автоматично використає Python альтернативу.

## Запуск

### Запуск Python сервера

```bash
python app.py
```

Сервер запуститься на `http://localhost:5000`

### Відкриття сайту

Відкрийте `index.html` в браузері або перейдіть на `http://localhost:5000`

## Функціонал

### Frontend
- Головна сторінка з описом послуг
- Форма замовлення з валідацією
- Адаптивний дизайн для мобільних пристроїв
- Плавна анімація при скролі

### Backend API
- `POST /api/order` - створення замовлення
- `GET /api/orders` - отримання всіх замовлень
- `GET /api/orders/<id>` - отримання замовлення за ID
- `PUT /api/orders/<id>/status` - оновлення статусу
- `GET /api/stats` - статистика замовлень

### C++ модуль
- Розрахунок ціни на основі типу послуги та складності
- Валідація email
- Розрахунок пріоритету замовлення
- Оцінка часу виконання

## Структура проекту

```
.
├── index.html              # Головна HTML сторінка
├── styles.css              # CSS стилі
├── script.js               # JavaScript для інтерактивності
├── app.py                  # Flask backend
├── order_processor.cpp     # C++ модуль
├── build.bat               # Скрипт компіляції для Windows
├── requirements.txt        # Python залежності
└── README.md              # Цей файл
```

## Використання

1. Відкрийте сайт у браузері
2. Заповніть форму замовлення
3. Виберіть тип послуги
4. Опишіть завдання
5. Вкажіть дедлайн та бюджет
6. Натисніть "Надіслати замовлення"
7. Система автоматично розрахує орієнтовну ціну
8. Замовлення збережеться у файлі `orders.json`

## Примітки

- Замовлення зберігаються у файлі `orders.json`
- Для повноцінної роботи рекомендовано встановити gcc/g++ для компіляції C++ модуля
- Сайт працює навіть без C++ модуля (з використанням Python альтернативи)
